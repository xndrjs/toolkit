## What xndrjs optimizes for

`xndrjs` is a small set of TypeScript libraries for applications that want architectural discipline without framework ceremony.

The core idea is simple: unknown data becomes useful only after it crosses an explicit trust boundary. Once that happens, your code can move validated, immutable, semantically named values through UI, orchestration, persistence, and transport with much less ambiguity.
